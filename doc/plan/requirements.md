# 기능 요구사항 정의서 (Requirements) - 암기빵 (Memory Bread)

## 1. 개요
'암기빵'은 사용자가 효율적으로 정보를 암기할 수 있도록 돕는 PWA 기반의 기억력 훈련 앱입니다. 인출 연습(Retrieval Practice), 간격 반복(Spaced Repetition), 청킹(Chunking) 전략을 활용합니다.

## 2. 주요 기능

### 2.1. 데이터 관리
- **학습 데이터 (Static):** JSON 형식으로 저장된 '키워드-설명' 쌍의 목록.
- **사용자 진행 상황 (Dynamic):** 브라우저 로컬 저장소(LocalStorage 또는 IndexedDB)에 저장.
    - 항목별 총 시도 횟수
    - 테스트 통과 횟수
    - 마스터 여부 (통과 횟수 3회 이상)
- **데이터 백업/복구:** 진행 상황 데이터를 텍스트 파일(JSON 등)로 내보내고 가져오는 기능.

### 2.2. 학습 페이즈 (Learning Phase)
- **청킹 전략:** 전체 항목 중 5~10개를 선택하여 한 세트로 구성.
- **플래시카드:** 
    - 앞면과 뒷면에 키워드 또는 설명을 노출.
    - 카드를 탭하면 뒤집히며 연관된 내용을 확인.
    - 양방향 학습: '키워드->설명' 또는 '설명->키워드' 무작위 노출.

### 2.3. 테스트 페이즈 (Test Phase)
- **객관식 퀴즈:** 4지 선다형.
- **문제 구성:** 학습 페이즈에서 다룬 항목들을 대상으로 함.
- **오답 생성:** 같은 묶음 내의 다른 항목의 설명을 무작위로 사용하여 보기 구성.
- **양방향 테스트:** '키워드 질문 -> 설명 답변' 또는 '설명 질문 -> 키워드 답변'.

### 2.4. 대시보드 및 진행 상황
- **묶음별 통계:**
    - 전체 항목 수
    - 학습 진행 항목 수
    - 테스트 통과 항목 수
    - 마스터 항목 수 (3회 이상 통과)

## 3. 기술적 요구사항
- **프레임워크:** Flutter (Web/PWA)
- **배포:** GitHub Pages (HTTPS 필수)
- **오프라인 지원:** Service Worker를 통한 PWA 지원.
- **UI/UX:**
    - 부드러운 카드 뒤집기 애니메이션.
    - 다크 모드 지원.
    - 반응형 디자인 (모바일, 태블릿 최적화).
