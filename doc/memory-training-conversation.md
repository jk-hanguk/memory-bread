## 1. GitHub Pages에서 PWA가 가능한가요?
**네, 아주 잘 됩니다.** PWA는 기본적으로 HTTPS 환경을 요구하는데, GitHub Pages는 기본적으로 HTTPS를 제공합니다.

- **설치 가능:** 브라우저 주소창 옆에 '앱 설치' 버튼이 생기며, 홈 화면에 아이콘이 추가됩니다.
- **오프라인 사용:** '서비스 워커(Service Worker)'라는 기술을 쓰면, 인터넷이 안 되는 지하철에서도 앱이 열리고 학습이 가능합니다.
- **배포 간편:** 코드를 수정해서 GitHub에 `push`만 하면 전 세계 어디서든 업데이트된 앱을 쓸 수 있습니다.

### B. 사용자 학습 진행 상황 (진행률, 통과 횟수)
사용자마다 다른 이 데이터는 브라우저 내부의 **Storage API**를 사용합니다.

## 4. 이 방식의 장점과 주의할 점

### 👍 장점

1. **완전 무료:** 서버 비용, DB 비용이 0원입니다.
